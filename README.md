# Simple ASTBuilder

### This is simple AST builder by java code.

## What's detecting:

* package
* class
* methods
* method parameters
* cycle for
* conditions 
  * if
  * else
  * break
  * continue
* declare variables
  * int a;
  * int a = 0;
  * int a = b + 10;
  * Object obj = new Object();  
* assigners
  * a = 26;
  * a = 26 - a;
  * a = 26 - a + b - 71;
* unary operators
  * increment ++
  * decrement --
* return statement
